## 📝 使用说明

### 配置插件

在 AstrBot 配置页面设置：
- **API 地址**：`http://localhost:8000`（你的 ncqq-manager 地址）
- **API 密钥**：从 ncqq-manager 获取
- **请求超时**：默认 10 秒
- **仅管理员**：默认 true

### 测试插件

1. 重启 AstrBot
2. 发送 `#ncqq状态` 查看是否连接成功
3. 发送 `#ncqq列表` 查看容器列表
4. 发送 `#ncqq帮助` 查看所有可用命令
